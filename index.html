Elementos de redes

El internet es una red; también lo es una computadora personal (PC) de escritorio conectada directamente a una laptop. Claramente, hay diferencias significativas entre los diferentes tipos de red. Un internet es una red de redes. Esta sección presenta un vocabulario para hablar sobre todos los tipos de red. Los sistemas en red a menudo se consideran conectados con la implicación de una conexión física, como la proporcionada por un cable que vincula los sistemas entre sí. El reciente auge de las tecnologías implementadas para la voz y la conexión en red de datos inalámbricos muestra que compartir una red solo requiere que los sistemas compartan un mecanismo mediante el cual puedan comunicarse.

Por el momento, podemos suponer que esos sistemas utilizan el mismo canal de comunicación (cualquiera que sea) y los mismos conjuntos de reglas (cualesquiera que sean) para gobernar sus transmisiones y recepciones. Sin embargo, en cualquier red en particular, también podemos suponer que solo hay un canal y un conjunto de reglas que rigen ese canal. También habrá límites en la cantidad de datos que se pueden transmitir en el canal y cuántos sistemas pueden comunicarse a través del mismo canal.

Antes de continuar, se necesitan algunas definiciones:

  • Entidad: Es un término amplio y genérico que puede referirse a cualquier objeto, elemento o sujeto que participa en un sistema o red. En el ámbito de las redes informáticas, las entidades pueden incluir dispositivos como computadoras, teléfonos móviles, procesos de software en ejecución, usuarios que interactúan con dispositivos, e incluso organizaciones enteras. El término "entidad" no se limita a elementos específicos de la red y puede referirse a una variedad de objetos, tanto dentro como fuera de la red.
  • Host: Un host es un tipo particular de nodo que envía o recibe datos en una red. En esencia, un host es un dispositivo que se conecta a una red para comunicarse con otros dispositivos en la red. Por ejemplo, computadoras de escritorio, dispositivos móviles, servidores, impresoras y otros dispositivos similares pueden considerarse hosts. Mientras que todos los hosts son nodos, no todos los nodos son hosts, ya que algunos nodos, como los enrutadores, actúan como dispositivos intermediarios en la red y no envían ni reciben datos directamente como hosts.
  • Nodo: Un nodo es un dispositivo o punto de conexión en una red. Los nodos pueden ser dispositivos finales, como computadoras, servidores y dispositivos móviles, o dispositivos de red intermedios, como enrutadores y conmutadores. En el contexto de una red informática, los nodos se identifican mediante direcciones IP únicas. A diferencia del término "entidad", que puede referirse a una amplia variedad de objetos, el término "nodo" se refiere específicamente a elementos que forman parte de la infraestructura de una red y que tienen una dirección IP.
  • Interfaz: Un punto de conexión en un nodo. Una interfaz puede ser cualquier cosa, desde un puerto de red en un servidor hasta una toma de teléfono en una pared. Una interfaz es esencialmente cualquier cosa que pueda conectarse a una red y que tenga una dirección MAC única.
  • Protocolo: Un conjunto de reglas que gobierna cómo los datos se transmiten y reciben en una red. Los protocolos pueden ser tan simples como HTTP o tan complejos como TCP/IP. Los protocolos son esenciales para que los dispositivos en una red puedan comunicarse entre sí.
  • Medio: El canal físico que se utiliza para transmitir datos a través de una red. El medio puede ser un cable, una fibra óptica, ondas de radio, etc.
Término Elementos informáticos para redes Descripción
Entidad - Usuarios Las personas que interactúan con la red, como navegando por Internet o usando aplicaciones en red.
- Procesos de software Programas o aplicaciones que se ejecutan en un dispositivo y que requieren comunicación en red, como un navegador web o un cliente de correo electrónico.
- Organizaciones Empresas, instituciones educativas u otras organizaciones que administran y utilizan redes informáticas.
- Dispositivos de red (nodos y hosts) Todos los dispositivos que forman parte de la red, incluidos nodos y hosts.
Nodo - Dispositivos finales Dispositivos como computadoras, servidores, dispositivos móviles e impresoras que tienen direcciones IP y participan en la comunicación en red.
- Dispositivos de red intermedios Equipos como enrutadores, conmutadores y puntos de acceso inalámbrico que ayudan a dirigir el tráfico de red y mantener la comunicación entre dispositivos finales.
Host - Computadoras de escritorio PCs que se conectan a una red para acceder a recursos compartidos, como archivos e impresoras.
- Dispositivos móviles Teléfonos inteligentes y tabletas que se conectan a redes Wi-Fi o celulares para acceder a Internet y aplicaciones en línea.
- Servidores Dispositivos potentes que almacenan y proporcionan recursos,como sitios web, archivos y aplicaciones, a otros dispositivos en la red.
- Impresoras Dispositivos que se conectan a una red para recibir trabajos de impresión de otros dispositivos, como computadoras y dispositivos móviles.
Interfaz - Puertos Ethernet Puntos de conexión físicos en dispositivos de red, como computadoras y enrutadores, que permiten la comunicación a través de cables Ethernet.
- Tarjetas de red inalámbricas Componentes internos de dispositivos como computadoras portátiles y dispositivos móviles que permiten la conexión a redes inalámbricas (Wi-Fi).
- Puertos USB Puertos en dispositivos que permiten la conexión de periféricos, como adaptadores de red USB, para habilitar la comunicación en red.
Protocolo - Protocolo de Control de Transmisión/Protocolo de Internet (TCP/IP) Conjunto de reglas que permite la comunicación entre dispositivos en Internet.
- Protocolo de transferencia de hipertexto (HTTP) Protocolo utilizado para solicitar y transmitir páginas web entre un navegador y un servidor web.
- Protocolo simple de transferencia de correo (SMTP) Protocolo utilizado para enviar correos electrónicos a través de Internet.
- Protocolo de configuración dinámica de host (DHCP) Protocolo utilizado para asignar automáticamente direcciones IP a dispositivos en una red.
Medio - Cable de par trenzado sin blindaje (UTP) Cable comúnmente utilizado para conectar dispositivos en una red Ethernet.
- Fibra óptica Medio de transmisión de alta velocidad que utiliza luz para transmitir datos a largas distancias.
- Ondas de radio (Wi-Fi) Medio inalámbrico que utiliza ondas de radio para transmitir datos entre dispositivos en una red Wi-Fi.
