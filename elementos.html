Elementos de una Red

  • Dispositivos finales (hosts): Computadoras de escritorio, portátiles, servidores, teléfonos móviles, impresoras y otros dispositivos que generan, consumen o procesan datos en la red.
  • Tarjetas de interfaz de red (NIC): Componentes de hardware en dispositivos finales y de interconexión que permiten la comunicación a través de la red, mediante cableado o conexiones inalámbricas.
  • Dispositivos de interconexión de red:
    • Hub (Capa 1): dispositivo simple que transmite señales eléctricas a todos los dispositivos conectados en una red.
    • Switch (Capa 2): dispositivo inteligente que conecta dispositivos en una LAN y permite la comunicación utilizando direcciones MAC.
    • Router (Capa 3): dispositivo que conecta y enruta paquetes entre diferentes redes utilizando direcciones IP.
    • Punto de acceso inalámbrico (WAP): dispositivo que permite la conexión inalámbrica de dispositivos a una red cableada.
  • Medios de transmisión: Cableado (cobre, fibra óptica), Inalámbrico (Wi-Fi, Bluetooth, tecnologías celulares).
  • Protocolos de red: Reglas y estándares que definen cómo se comunican los dispositivos en la red y cómo se transmiten y procesan los datos (por ejemplo, TCP/IP, Ethernet, Wi-Fi).
  • Direcciones IP y MAC:
    • Direcciones IP: identificadores únicos asignados a dispositivos en una red que utilizan el protocolo IP para la comunicación.
    • Direcciones MAC: identificadores únicos asignados a las tarjetas de interfaz de red que permiten la comunicación a nivel de enlace de datos (por ejemplo, Ethernet).
  • Software de red: Aplicaciones, sistemas operativos y servicios que facilitan la comunicación y el intercambio de datos en la red (por ejemplo, navegadores web, correo electrónico, servidores web).
  • Equipos de seguridad de red:
    • Firewalls: dispositivos o software que filtran y controlan el tráfico de red para proteger los dispositivos y datos de amenazas y accesos no autorizados.
    • Sistemas de detección y prevención de intrusiones (IDS/IPS): dispositivos o software que monitorean la red en busca de actividades sospechosas o maliciosas y toman medidas para proteger la red.
  • Entidad: se refiere a cualquier objeto o elemento dentro de la red que tenga una identidad única y pueda comunicarse con otros elementos.
  • Host: son dispositivos finales que generan, consumen o procesan datos en la red, como computadoras de escritorio, portátiles, servidores, teléfonos móviles, impresoras y otros dispositivos..
  • Nodo: se refiere a cualquier dispositivo dentro de la red que tenga capacidad de procesamiento, almacenamiento y comunicación, incluyendo tanto dispositivos finales como dispositivos de interconexión de red. .
  • Interfaz: son componentes de hardware, como las tarjetas de interfaz de red (NIC), que permiten la comunicación entre dispositivos mediante cableado o conexiones inalámbricas.
  • Protocolo: son reglas y estándares que definen cómo se comunican los dispositivos en la red y cómo se transmiten y procesan los datos. Ejemplos de protocolos son TCP/IP, Ethernet y Wi-Fi.
  • Medio: son los medios de transmisión utilizados para la comunicación entre dispositivos, como el cableado (cobre, fibra óptica) y las conexiones inalámbricas (Wi-Fi, Bluetooth, tecnologías celulares).
